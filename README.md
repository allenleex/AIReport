## 项目背景

xLua为Unity、 .Net、 Mono等C#环境增加Lua脚本编程的能力，借助xLua，这些Lua代码可以方便的和C#相互调用。

## 工程结构

* aireport-admin 后台管理系统（Java语言，SpringMVC+JDBC框架）
* aireport 爬虫系统（Python语言，Scrapy框架）
* aireportdemo 爬虫范例，采集数据写入MySQL数据库（Python语言，Scrapy框架）
